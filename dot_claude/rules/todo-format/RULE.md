# TODO Comment Format

When writing TODO comments in any language, use a single-line comment with this format:

```
TODO: [vruby YYYY-MM-DD] <description>
```

- **Author**: Always `vruby`
- **Date**: Current date in `YYYY-MM-DD` format unless a specific date is provided
- **Description**: Clear, actionable description of what needs to be done

**Example:**
```typescript
// TODO: [vruby 2026-03-24] Replace with dedicated CHAT role once available
```
